# SignalLane CLI

SignalLane is intentionally small and scriptable.

## Entry points

- `signallane`
- `signallane --version`
- `signallane --help`

## Commands

- `signallane init`
- `signallane doctor`
- `signallane reply`
- `signallane reply --from ./post.txt`
- `signallane improve --text "..."`
- `signallane score --text "..."`
- `signallane angles --topic "..."`
- `signallane hooks --topic "..."`
- `signallane profile set`
- `signallane profile show`

## TUI

Running `signallane` with no arguments opens a simple terminal menu.

The interface stays black and white, works in standard terminals, and keeps keyboard controls simple.
