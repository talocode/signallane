# Local-First Design

SignalLane v0.1.0 does not require hosted services.

## What is local

- Profile data
- Drafts
- History
- All generation logic

## What is not included

- X login
- Cookies
- Scraping
- Auto-posting
- Cloud sync
- Paid AI dependencies

## Storage

Default storage is:

```text
~/.signallane/
```

Use `SIGNALLANE_HOME` to redirect storage for testing or portable setups.
