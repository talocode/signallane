Install

Run with npx

npx @talocode/signallane

Global install

npm install -g @talocode/signallane signallane

What it does

Write sharper replies from pasted posts

Turn a post, thread, reply draft, or local text file into something clearer and more human.

Improve draft replies

Trim robotic phrasing, keep the voice human, and make the reply easier to read.

Score replies for clarity and human tone

See where a draft feels strong and where it risks sounding generic or AI-like.

Generate content angles

Explore contrarian, founder, technical, pain, and story angles for a topic.

Generate hooks

Get short hook ideas without clickbait or fake claims.

Store a local builder profile

Keep your profile on your machine and reuse it when shaping replies and ideas.

No X login required

SignalLane stays local-first and does not ask for credentials or cookies.

No hosted service required

Use it directly from the terminal without depending on a cloud backend.

Commands

signallane init
signallane doctor
signallane reply
signallane improve --text "your draft reply"
signallane score --text "your reply"
signallane angles --topic "open source AI tools"
signallane hooks --topic "AI coding agents"
signallane profile show

Local-first

  • No secrets required
  • No telemetry
  • No auto-posting
  • You stay in control
